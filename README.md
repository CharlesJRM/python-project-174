# gendiff

**gendiff** es una herramienta de línea de comandos y librería de Python que compara dos archivos (JSON o YAML) y muestra sus diferencias de manera estructurada.

## Características
- Compara archivos JSON y YAML.
- Soporte para diferentes formatos de salida (estilizada, plana, JSON).
- Se puede usar tanto desde la terminal como dentro de scripts de Python.

---

## 1. Instalación

### Clonar el repositorio
```bash
git clone https://github.com/CharlesJRM/python-project-140.git
cd python-project-140

2. Instala las dependencias usando Poetry:

poetry install

Opciones disponibles

-f, --format [stylish|plain|json] — Formato de salida de las diferencias.

Si usas pip:

pip install -r requirements.txt


3. Uso desde la terminal

Ejecuta el comando comparando dos archivos:

poetry run gendiff file1.json file2.json


También puedes especificar un formato de salida:

poetry run gendiff --format plain file1.json file2.json
poetry run gendiff --format json file1.yaml file2.yaml


Opciones disponibles:

--format stylish (por defecto)

--format plain

--format json
